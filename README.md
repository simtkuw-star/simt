# سِمْط

سِمْط منصة أدبية تربوية تفاعلية تهدف إلى تنمية مهارات الكتابة الإبداعية لدى طالبات المرحلة الثانوية، من خلال تجربة تجمع بين معسكر الكتابة، محرر النصوص، ومعجم سِمْط.

## فكرة المشروع

يساعد مشروع سِمْط الطالبة على بناء المقال خطوة بخطوة، من توليد الفكرة إلى تنظيم الفقرات والتحرير النهائي، مع المحافظة على صوتها الخاص وأسلوبها العربي الأصيل.

## مكونات المنصة

- معسكر سِمْط لتنمية مهارات الكتابة.
- محرر كتابة تفاعلي لحساب الكلمات والحروف.
- معجم سِمْط للمعاني والمترادفات والأضداد.
- مساحة مجلة لعرض إبداعات الطالبات.
- أسئلة شائعة تساعد الزائرات على فهم المنصة.
